Insert Email

Enter a javascript expression and click save to add it to the list of options inserted when you hit the insert email hotkey.

It's a JavaScript expression so if you want just a plain string you need to put it in quotes. e.g. 'mycommonusername' or "mycommonusername".

window.location.host.split('.').slice(-2,-1)[0] + '@example.com' will add the domain name of the current site plus @example.com. i.e. on amazon.com it would add amazon@exmaple.com.
